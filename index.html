GainForge

HDR ゲインマップ付き JPEG を、ゲインマップを保持したまま HEIC に変換する macOS アプリ。写真アプリと同じ HDR 構造で書き出します。

⬇︎ ダウンロード GitHub で見る

macOS 15 以降 ・ Apple 署名 / 公証済み ・ 無料 ・ 自動更新対応

特長

写真アプリ書き出しと画素レベルでほぼ一致する HDR 変換

ゲインマップを生転写

元のカラーゲインマップを Display P3 PQ のまま転写。SDR ベース画像 + ISO ゲインマップで、写真アプリと同じ HDR 構造を保ちます。

一括変換

JPEG / フォルダをドラッグ&ドロップ。コア数に応じて複数枚を並列変換し、1 件ごとに結果をライブ更新します。

比較ビューワ内蔵

変換前後を並べて、HDR 表示のまま画質を確認できます。

メタデータ保持

EXIF / GPS / 向き（Orientation）などの撮影情報を引き継いで書き出します。

インストール

ダウンロードして「アプリケーション」に入れるだけ

ダウンロード上の「ダウンロード」ボタンから zip を入手します。
展開zip をダブルクリックすると GainForge.app ができます。
移動GainForge.app を「アプリケーション」フォルダにドラッグします。
起動ダブルクリックで起動。初回に確認が出たら「開く」を押してください（公証済みのため警告ブロックは出ません）。
